Specs

The Game Boy Advance was much more powerful than its predecessors. Instead of an 8.38 MHz Z80 processor like the Game Boy Color did, the GBA had a 16.8 MHz ARM processor. NOW YOU'RE PLAYING WITH POWER! However, the Z80 processor is included in most Game Boy Advance models for backwards compatiblity. The Game Boy Advance had 256kb of RAM on board. The screen resolution is 240 pixels by 160 pixels, which is higher resolution in both dimensions than its predecessors. Like the Game Boy Color, it uses a 15-bit color palette. The Game Boy Advance uses a dual 8-bit DAC for sound. All Game Boy Advance models had shoulder triggers, aka L & R buttons. Game Boy Advance cartridges physically were about half of the size as original Game Boy & Game Boy Color cartridges.

Models

Game Boy Advance (AGB)

The first out of the 4 official Game Boy Advance models! It was released in 2001, 4 years after the release of the Game Boy Color. The main difference from its predecessors is its chassis. Instead of putting controls below the screen, the controls were put next to each side of the screen. This created a horizontal form factor, which was very comfortable to use. This was the last Game Boy to use AA batteries, and a screen without a light.

Game Boy Player

This model is the successor to the Super Game Boy. This device allows you to play Game Boy Advance and prior games on your TV. Instead of plugging into a Super Nintendo, the Game Boy Player plugged into the bottom of the GameCube. Like the Super Game Boy, the Game Boy Player has offical Game Boy hardware, just without buttons and a screen. Instead of Game Boy hardware though, the Game Boy Player has Game Boy Advanced hardware.

In order to initialize the Game Boy Player on startup, the GameCube needed a special boot disc. The Game Boy Player suffers from the same problem that Inkjet printers do. Inkjet printers are cheap, but ink cartridges are expensive. Although the Game Boy Player is cheap, the boot disc is somewhat expensive on the used market. However, the Game Boy Player boot disc is unecessary if your GameCube is running unoffical homebrew software.

Game Boy Advance SP Frontlight (AGS-001)

This model was relased only 2 years after the original, for multiple reasons. The most odvious reason was to add a lighting system. People were getting tired of the reflective screen that Game Boys have had previously. While the reflective layer in the display is still present, a frontlight was added to the display. This made playing on the device tolerable, although it isn't as nice as a backlight display for multiple reasons.

The other main difference is the form factor. The design was similar to a flip phone, but a bit wider. This has multiple advantages, as the devices screen and controls were protected when the device was closed. The device was also smaller than a regular Game Boy Advance when closed. The main disadvantage was ergonomics, as some people hated wrapping their hands around a tiny square.

The final difference is the battery. Instead of using AA batteries, it used a rechargeable battery. This is an extremely convenient feature, although some people disliked it because it used a proprietary charging connector to charge. Because the charging port was on the top of the unit, the cartridge contacts are on the bottom of the unit.

Game Boy Advance SP Backlight (AGS-101)

This model shared the exact same chassis as its older brother AGS-001. The main difference in this model is the screen. The screen uses backlight instead of a frontlight. This has multiple advantages, as the backlight has much higher contrast and brightness. These models sell for a bit more on the used market, but I highly suggest this model if you are planning to buy a Game Boy.

This model removes the reflective layer, so you can't use an external source of light (such as the sun) with this unit. Like other backlight displays, external light will only make the display look more washed out.

Left is the AGS-001 (frontlight) & the right is the AGS-101 (backlight).

Game Boy Micro (GBM)

This is one of the most odd Game Boys in the line up. It was released in 2005, and is the last official Game Boy. This model is really small, hence the name. The best way to describe it is an original Game Boy advance that went through a shrink ray. It shares the same horizontal form factor, but the screen is only 2 inches diagonal. Like the AGS-101, the display has a backlight. It is also more rectangular than the original model.

Instead of a plastic build, the Game Boy Micro has a metal build. For a Nintendo handheld, a metal handheld isn't common. Even the Nintendo Switch chassis is made out of plastic. The likely reason Nintendo used metal was to keep the device durable while being small.

Instead of a screen lens to protect the screen, face plates were used. The main difference is that screen lenses are held down by adhesive, while face plates are held down by special clamps. This made face plates easy to swap.

Like the SP, cartridge contacts are at the bottom of the unit while the charging port is on top. No link port is on the model, but an adapter can be used to bring it back.

Before buying this model, please know that this console can only play Game Boy Advance games. The contacts for Game Boy and Game Boy color games are non-existant, so they can't be played natively.
