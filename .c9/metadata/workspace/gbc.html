Types of Games

\n
\n

There is a lot of confusion on the different types of Game Boy Color games, so I figured I would explain the differences of each.

\n
    \n
  • Game Boy Games: These games are compatible with all Game Boy systems, and the Game Boy Color could also play them in a backwards compatibility mode.
    • \n
  • Game Boy Color Enhanced Games: These games were designed for the Game Boy Color, but could operate on a Game Boy with a 2-bit color palette instead.
    • \n
  • Game Boy Color Exclusive Games: These games took advantage of the faster hardware of the Game Boy Color, so they could not operate on a regular Game Boy.
    • \n
\n

Game Boy games came in a gray cartridge, Game Boy Color Enhanced games came in a black cartridge, and Game Boy Color Exclusive games came in a transparent convex cartridge.

\n \n
\n \n
\n

Specs

\n
\n

The Game Boy Color has very similar specifications as the original Game Boy. The only things different are the clock speed and screen. The clock speed is 8.38 MHz, which is twice as much as the original Game Boy. The screen is the same resolution as the old model, but offers a 15-bit color palette instead of the 2-bit color palette.

Models

\n
\n

Game Boy Color (GBC)

\n \n

This was the only model with Game Boy Color hardware. It was released in 1998, 2 years after the Game Boy Pocket. It featured a chassis slightly bigger to the Game Boy Pocket. Instead of 2 AAA batteries like the Game Boy Pocket did, it used 2 AA batteries. In order to fit the AA batteries, the back of the Game Boy Color is slightly convex.

\n

The other main difference in this particular model is the screen. It was physically smaller than the Game Boy & Game Boy Pocket screen. Because it used a different display technology, no noticable ghosting can be found in this model.

\n

Excluding special editions, the Game Boy Color shell had 6 color options: berry, grape, kiwi, dandelion, teal, and atomic purple. Atomic purple is the only shell color that was partially transparent.
